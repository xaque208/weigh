# Weigh

A small ruby utility to wieigh directory contents.  Heavy stuff to the bottom.

## Example output

        7 bytes   ./.gitignore
       90 bytes   ./bin/
      343 bytes   ./README.md
      670 bytes   ./weigh.gemspec
       4.90 KiB   ./lib/
      28.38 KiB   ./.git/
             --- ---
      34.37 KiB   :total size
             --- ---

## Installation

    gem install weigh
